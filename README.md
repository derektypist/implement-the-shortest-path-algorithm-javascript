# Implement the Shortest Path Algorithm (JavaScript)

Implement the Shortest Path Algorithm in JavaScript.  Compute the shortest path between nodes in a weighted graph and returns both the distances and the paths taken.

## Technologies

Uses JavaScript.

## Deployment

Copy `script.js` to the text editor of your choice.  If using terminal, use the command `node script.js` to run the code.
